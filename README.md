# PascalCTF 2026 üö©
The **Pascal CTF** is a *cybersecurity event* organized by a technical team composed of various current and former students from [`ITT Blaise Pascal`](https://ispascalcomandini.it/) in Cesena all part of the CTF team [`Paolo`](https://ctftime.org/team/263145).

This event aims to increase knowledge and curiosity about cybersecurity, promoting its development.
